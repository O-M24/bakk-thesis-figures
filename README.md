# Thesis Figures — Full Resolution

**Vertical Traceability in Multi-Domain Warehouse Systems: A Hierarchical procan.do Approach**
Bachelor thesis, Oliver Mähr — TU Wien, Media Informatics and Visual Computing.

This repository holds the figures of the thesis in full resolution, so the diagrams can be viewed at a legible size. Several figures are detailed PAN/CEN diagrams whose text becomes small when the figure is scaled to fit the thesis page.

- **`figures/`** — all diagrams as PNG at full resolution.
- **`sources/`** — editable [draw.io](https://app.diagrams.net) sources for selected diagrams. Open a `.drawio` file at app.diagrams.net and zoom in for sharp text at any size.
